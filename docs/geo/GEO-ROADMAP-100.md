## Resumo: o que é código vs. o que é decisão de negócio

### Posso implementar agora (código)
- Página de Política de Privacidade (template)
- Informações de contato no footer (endereço, WhatsApp, e-mail)
- Imagens responsivas com `srcset`
- Ajuste de fonte base para 16px
- Preload hints para fontes
- Schema `speakable`
- IndexNow para Bing
- Template de página em inglês

- Depoimentos reais de clientes
- Conteúdo expandido de /servicos e /sobre
- 20-30 artigos para o blog
- Decisão sobre branding "Arven Aquisição"

### Precisa de ação externa (plataformas)
- Wikipedia / Wikidata
- YouTube (criar canal, gravar vídeos)
- Reddit, LinkedIn (presença ativa)
- Clutch, G2, Capterra (criar perfis, pedir reviews)
- Google Business Profile (criar + verificar)
- Bing Webmaster Tools (verificar domínio)
- Reclame Aqui (criar perfil)
